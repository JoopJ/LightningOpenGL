# LightningOpenGL
An exploration of the methods used to create realistic looking and behaving lightning

## Uses
- glad-4.6
- glfw-3.3.8
- glm-0.9.9.8
- imgui-1.89.3

## How to use

Space - Recalculates the lightning pattern.

W/A/S/D - Move the camera
